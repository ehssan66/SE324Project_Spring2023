# Server

This directory contains the server side to the phase one of the project.

## Run the code

To run, type the following command to build the project.
Note that you need to have maven installed on your machine, and you need java version 19.
```
mvn clean install
```
Then go to src/main/java/com/alaeddin/server/Application.java and run the main class.
